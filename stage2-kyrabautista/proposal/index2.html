About

INK-ki is an interactive illustration web app inspired by Inktober, a popular, worldwide illustration event. Every year, artist from around the world produce illustrations based on a prompt list for every day of the month. Just like Inktober, INK-ki provides users with a list of prompts and "ink" (so there won't be any erasing!), however this time, the user can choose up to three prompts for one illustration. The challenge that INK-ki provides is based on creating an illustration using all three prompts.

Data from https://www.datamuse.com/api/
Canvas features from http://www.williammalone.com/articles/create-html5-canvas-javascript-drawing-app/ & YouTube tutorials.
